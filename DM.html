Distance Measurement System

Infra red remode controled distance object identifying system with luminosity

Step By Step Explaination

  • Initialize components including LCD, IR receiver, ultrasonic sensor, LEDs, and buttons.
  • Detect objects within the range using the ultrasonic sensor and trigger it periodically.
  • Process and display distances in either centimeters or inches on the LCD.
  • Set warning LED blink rate based on the distance from objects.
  • Lock the system and show warnings when an object is too close.
  • Use an IR remote to control functions like unlocking, unit conversion, and LCD mode switching.
  • Monitor luminosity with a photoresistor and adjust the light LED accordingly.
  • Unlock the system by pressing a button when locked.
  • Reset settings to default through the IR remote.
  • Continuously loop through these operations to maintain functionality.
